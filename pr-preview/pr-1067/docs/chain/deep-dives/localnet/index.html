Localnet

Learn how to deploy localnet of Push Chain to understand or speed up the development of your app.

Prerequisites‚Äã

Ensure the following tools are installed:

  • Go (v1.21+)
  • Foundry
  • Git

Setup & Start Push Chain Localnet‚Äã

1. Clone and install Push Chain‚Äã

git clone https://github.com/pushchain/push-chain
cd push-chain
make install


2. Start the local testnet‚Äã

make sh-testnet

This launches the local Push Chain node. You can now interact with it using the pchaind CLI.

Deploy Protocol Contracts (UEA)‚Äã

To enable universal execution from EVM/Solana chains, you need to deploy protocol contracts to your localnet.

1. Clone and build smart account contracts‚Äã

git clone https://github.com/pushchain/push-smart-account-v1
cd push-smart-account-v1
forge build


2. Fund Your EVM Wallet on Localnet‚Äã

Before deploying the contracts, your EVM wallet must have enough UPC tokens on the local Push Chain network.

To send funds, you need the Bech32 format of your EVM wallet address (since Push Chain uses Cosmos SDK-style addresses for transfers).

Step-by-step‚Äã

  1. Take your EVM address (in hex, 0x-prefixed format) ‚Äî for example:
0x778d3206374f8ac265728e18e3fe2ae6b93e4ce4
  1. Convert it to Bech32 format using:
pchaind debug addr 778d3206374f8ac265728e18e3fe2ae6b93e4ce4

üîÅ Replace 778d...ce4 with your own EVM address (without the 0x prefix).

  1. You should get an output like this:
Address (hex): 778D3206374F8AC265728E18E3FE2AE6B93E4CE4
Bech32 Acc: push1jtdw9kjc2yptl6yjyad69q73v2gcl29xfmmq5a
  1. Use the Bech32 account address (e.g. push1jtdw9kjc2yptl6yjyad69q73v2gcl29xfmmq5a) to receive funds:
pchaind tx bank send push1gjaw568e35hjc8udhat0xnsxxmkm2snrexxz20 push1jtdw9kjc2yptl6yjyad69q73v2gcl29xfmmq5a 100000000000000000000000upc --gas-prices 1000000000upc -y

The push1gjaw568e35hjc8udhat0xnsxxmkm2snrexxz20 address is pre-funded at genesis with 2 billion UPC, so it‚Äôs safe to use as the sender.



3. Deploy UEAFactory & Implementations‚Äã

Once funded, run the deployment script:

forge script script/deployFactory.s.sol --rpc-url http://localhost:8545 --broadcast --private-key <YOUR_LOCAL_PRIVATE_KEY>

This sets up the UEAFactory and registers the EVM + SVM UEA implementations needed for universal execution.

Add External Chain Configurations‚Äã

Now configure Push Chain to interact with external chains.

Solana Devnet‚Äã

pchaind tx ue update-chain-config --chain-config '{
  "namespace": "solana",
  "chain_id": "EtWTRABZaYq6iMfeYKouRu166VU2xqa1",
  "public_rpc_url": "https://api.devnet.solana.com",
  "vm_type": 1,
  "locker_contract_address": "3zrWaMknHTRQpZSxY4BvQxw9TStSXiHcmcp3NMPTFkke",
  "block_confirmation": 1,
  "funds_added_event_topic": "FundsAddedEvent"
}' --from acc1 --gas-prices 1000000000upc -y


Ethereum Sepolia‚Äã

pchaind tx ue add-chain-config --chain-config '{
  "namespace": "eip155",
  "chain_id": "11155111",
  "public_rpc_url": "https://1rpc.io/sepolia",
  "vm_type": 0,
  "locker_contract_address": "0xdEF1a1265247488e5934706657404F40A81eDEe4",
  "block_confirmation": 6,
  "funds_added_event_topic": "0xddcd6ef7998ae51b4ead4e9aa669a7d5ff30af88eddaa5062c91b08153da07c0"
}' --from acc1 --gas-prices 1000000000upc -y

If you‚Äôve deployed a custom FeeGateway contract, you can include its address in the chain config as well.

That‚Äôs it! You have successfully set up the Push Chain localnet and are ready to bring your imagination into reality.
